# Фаза 2 — Подключение к серверу

## Статус
- Статус: ✅
- Дата/время: 2026-04-07 07:36:21 +05:00

## Цель фазы
- Настроить устойчивый SSH-доступ к серверу по ключу, чтобы дальнейшие фазы выполнялись без ручного ввода пароля.

## Что сделано
- Сгенерирован локальный SSH-ключ `~/.ssh/voice-inbox-server`.
- Выполнено первое подключение к серверу `94.103.178.210` под `root` через `paramiko`.
- Публичный ключ добавлен в `/root/.ssh/authorized_keys`.
- Host key сервера сохранён в локальный `~/.ssh/known_hosts`.
- В локальный `~/.ssh/config` добавлен alias `voice-inbox-server`.
- Проверен обычный SSH-вход по alias.

## Проблемы и отклонения
- На сервере root-пароль оказался истёкшим и требовал немедленной смены.
- Попытка пройти forced password change без TTY через `exec_command()` не сработала.
- Попытка оставить тот же пароль не прошла: сервер потребовал действительно новый пароль.
- После настройки alias `ssh.exe` на Windows отказался читать `~/.ssh/config` из-за лишних ACL.
- Первая версия локального ключа оказалась фактически запароленной из-за неудачного quoting при `ssh-keygen`, поэтому сервер принимал public key, но аутентификация по приватному ключу не завершалась.

## Как решено
- Через `paramiko.invoke_shell()` был открыт интерактивный PTY и выполнена forced password change.
- Для root сгенерирован новый случайный пароль; локальный `D:\vapecoding\intFinDop\.env` синхронизирован с новым `SERVER_PASSWORD`.
- ACL на `C:\Users\avp-h\.ssh\config` были исправлены через `icacls`.
- Неправильно сгенерированный ключ удалён и перегенерирован уже с действительно пустым passphrase.
- Новый public key повторно добавлен в `/root/.ssh/authorized_keys`.

## Проверки
- Новый пароль root проходит через `paramiko`, старый больше не работает.
- Host key сервера сохранён локально.
- `ssh voice-inbox-server "echo ok"` прошёл успешно.
- Дополнительно подтверждено `ssh voice-inbox-server "hostname && whoami"` → `box-906559` / `root`.

## Артефакты и изменения
- Созданы/обновлены локальные файлы:
  - `C:\Users\avp-h\.ssh\voice-inbox-server`
  - `C:\Users\avp-h\.ssh\voice-inbox-server.pub`
  - `C:\Users\avp-h\.ssh\config`
  - `C:\Users\avp-h\.ssh\known_hosts`
  - `D:\vapecoding\intFinDop\.env`
- Обновлён серверный `/root/.ssh/authorized_keys`.
- Raw пароль root в отчёт не записывался.
